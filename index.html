Linha do tempo do projeto de
Rafael jesus

Feito na NLW da Rocketseat
15 de Março de 2023

Primeiro dia de evento

No primeiro dia, as primeiras linhas de código em HTML foram criadas, adicionando titulos e imagens ao site.

16 de Março de 2023

Segunda aula

Na segunda aula, criamos o arquivo CSS para moldar a estrutura visual do site, adicionando o background e decidindo o lugar de cada elemento no grid.

17 de Março de 2023

Terceira aula

Neste dia, continuamos desenvolvendo a parte do código em CSS, aprimorando o layout e as fucnionalidades do site.

18 de Março de 2023

Quarta aula

Na penúltima aula do evento, aprendemos como linkar as imagens e textos com outros sites (redes sociais por exemplo), e aplicamos a sub-função hover para deixar o site menos estático, mas não consegui aplicá-la muito bem, então só estará presente na parte direita do site. Tambem fizemos a parte responsiva do código.

19 de Março de 2023

O resultado final

Na última aula do evento, aprendemos a usar o git e o github para upar nosso projeto para um diretório.
